# LINE Official Account - Nissy Beauty Salon

美容室「Nissy」のLINE公式アカウントカスタマイズプロジェクト

## 📋 プロジェクト概要

美容室のLINE公式アカウントを活用し、顧客とのコミュニケーションを充実させるためのカスタマイズプロジェクトです。

## ✨ 実装機能

### LINEリッチメニュー構成（6分割）

1. **お知らせ（information）** 📧
   - キーワード応答による多階層メニュー
   - 季節のご案内、キャンペーンなどを不定期配信

2. **ご予約（reserve）** 📅
   - 一旦は既存予約アプリ「STEKiNA」へリンク
   - URL: `https://stekina.com/profile/8CY4-G2Z4?stype=lp`
   - 追々変更の可能性あり

3. **こだわり（commitment）** 💭
   - 現在準備中、後日追加予定

4. **動画（Video）** 📹
   - YouTubeチャンネルへのリンク
   - 商品の使い方動画（撮影中）
     - イオンシャワーの使い方
     - ホホバオイルのスゴイトコ
     - 季節のお手入れ方法
     - 商品購入時の注意点

5. **商品（item search）** 🛍️
   - 取り扱いブランドの紹介・公式サイトへの導線ページ
   - 詳細は下記「商品ページの方針」を参照

6. **Instagram** 📸
   - 公式Instagramアカウントへのリンク

## 📱 LINE公式アカウント設定

### キーワード応答設定

LINE公式アカウントの管理画面から、以下のキーワード応答を設定します:

**お知らせメニュー:**
- トリガー: リッチメニュー「お知らせ」エリアタップ
- 応答: 選択肢画像（新着情報/営業日/キャンペーン）
- 各選択肢でさらに詳細情報を表示

**こだわりメニュー:**
- トリガー: リッチメニュー「こだわり」エリアタップ
- 応答: 選択肢画像（使用商品/施術へのこだわり/サロンの雰囲気）
- 各選択肢でさらに詳細情報を表示

- 現行の予約管理アプリ

- LINEリッチメニューのキーワード応答を多重入れ子で活用
- 商品ページは段階的リリース（フェーズ1: リンク集モック → フェーズ2: 本番カタログシステム）
- 動画コンテンツはYouTubeで管理
- 決済機能は不要（来店時に対面決済）
- サーバーは友人の案件（`oshirase`アカウント）と共用中のため、恵専用のサーバー契約は本番化のタイミングで検討
- モック公開はGitHub Pages、本番はCloudflare Pagesを候補として検討中
