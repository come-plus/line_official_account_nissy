# LINE Official Account - Nissy Beauty Salon

美容室「Nissy」のLINE公式アカウントカスタマイズプロジェクト

## 📋 プロジェクト概要

美容室のLINE公式アカウントを活用し、顧客とのコミュニケーションを充実させるためのカスタマイズプロジェクトです。

## ✨ 実装機能

### LINEリッチメニュー構成（6分割）

1. **お知らせ** 📧
   - キーワード応答による多階層メニュー
   - 新着情報、営業日、キャンペーン情報など

2. **ご予約** 📅
   - Square予約システムへのリンク
   - 予約確認・変更案内

3. **こだわり** 💭
   - キーワード応答による多階層メニュー
   - サロンのこだわり、使用商品の説明など

4. **動画** 📹
   - YouTubeチャンネルへのリンク
   - 商品の使い方動画
     - イオンシャワーの使い方
     - ホホバオイルのスゴイトコ
     - 季節のお手入れ方法
     - 商品購入時の注意点

5. **商品** 🛍️
   - 商品注文システムへのリンク
   - 詳細は → [hair_salon_order_system](https://github.com/come-plus/hair_salon_order_system)

6. **Instagram** 📸
   - 公式Instagramアカウントへのリンク

## 📱 LINE公式アカウント設定

### キーワード応答設定

LINE公式アカウントの管理画面から、以下のキーワード応答を設定します:

**お知らせメニュー:**
- トリガー: リッチメニュー「お知らせ」エリアタップ
- 応答: 選択肢画像（新着情報/営業日/キャンペーン）
- 各選択肢でさらに詳細情報を表示

**こだわりメニュー:**
- トリガー: リッチメニュー「こだわり」エリアタップ
- 応答: 選択肢画像（使用商品/施術へのこだわり/サロンの雰囲気）
- 各選択肢でさらに詳細情報を表示
